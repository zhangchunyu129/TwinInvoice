# TwinInvoice

TwinInvoice 是一个轻量的浏览器发票合并打印工具，可将 PDF 和图片发票自动排版到 A4 页面中，并支持打印或下载合并后的 PDF。适合二合一的发票会放在同一张 A4 纸上，高度不适合的发票会自动单独占用一张 A4。

## 功能

- 支持导入单个或多个发票文件，文件可多选。
- 支持导入整个发票文件夹。
- 支持递归读取子文件夹。
- 支持 PDF、PNG、JPG、JPEG、WEBP 文件。
- 按文件路径排序后生成打印预览。
- 自动将两张适合的发票合并到一张 A4 纸。
- 遇到单数张适合二合一的发票时，继续向后配对，避免中间出现只有一张发票的 A4 页。
- 自动将过高或不适合二合一的发票单独排版。
- 合并后的 A4 页默认全选，可按整页取消打印或下载。
- 支持将选中的 A4 页下载为一个 PDF 文件。
- 使用本地 PDF.js，在浏览器中直接运行。

## 使用方式

1. 用浏览器打开 `invoice-print.html`。
2. 选择一个或多个发票文件，或选择整个发票文件夹。
3. 确认生成的 A4 打印预览。每张 A4 页默认选中，不需要的整页可取消勾选。
4. 点击“打印”打印选中的 A4 页，或点击“下载”生成包含选中页的 PDF 文件。
